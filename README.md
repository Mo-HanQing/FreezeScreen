## English

## ğŸ’¡ Why do I need it?

In daily work or professional workflows, you may need to step away from your computer, or prevent external factors (pets, kids, or cleaning the screen/keyboard) from interfering with the current state.

**FreezeScreen** helps you:

- **Prevent file corruption**: Lock input during long file transfers, rendering, or compilation to avoid interruptions caused by accidental touches.
- **Protect your workspace**: Prevent others (or pets) from moving the mouse / typing on the keyboard, which could lead to data loss or unwanted changes.
- **Safe cleaning**: Wipe the keyboard/screen without shutting down.
- **Stable presentation**: Keep the screen steady when presenting static content.

## âœ¨ Features

- **Global lock**: Blocks keyboard and mouse input; unlock with the correct password.
- **Always on top**: The lock window stays topmost to reduce accidental window operations.
- **Change password**: Supports custom password; default is `1234`.
- **Lightweight & reliable**: Built with JavaFX + JNativeHook.
